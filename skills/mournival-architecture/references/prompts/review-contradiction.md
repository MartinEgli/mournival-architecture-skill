# Review Contradiction

Use when two or more claims, sources, versions, or contexts conflict.

## Required Checks

- Identify conflicting statements.
- Identify source and version for each statement.
- Compare evidence class and execution context.
- Decide whether conflict needs ADR, context split, escalation, or rejection.
- Preserve unresolved conflict when evidence is insufficient.

## Steward Questions

Decision: Is decision, ADR, context split, or escalation needed?

Risk: What harm follows from wrong resolution?

Value: How should the contradiction be documented for users?

Assurance: Which statement is better supported, and by what evidence?
