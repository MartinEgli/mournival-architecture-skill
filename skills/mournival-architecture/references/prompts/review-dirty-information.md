# Review Dirty Information

Use when reviewing uncertain, contaminated, partial, or unverified information.

## Required Checks

- Identify what is known.
- Identify what is missing.
- Identify possible source request.
- Decide whether it can become a hypothesis or candidate.
- Prevent productive use until review state, evidence class, and traceability
  exist.

## Steward Questions

Decision: Should it become Source Request, Hypothesis, KCC, or Reject?

Risk: How dangerous is false adoption?

Value: Is it useful enough to investigate?

Assurance: Which evidence is missing, and may a candidate exist yet?
