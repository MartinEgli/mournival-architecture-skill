# Review Clean AI

Use when reviewing AI-generated, AI-assisted, or AI-dependent architecture
claims and artifacts.

## Required Checks

- Identify whether content came from human source, AI output, or mixed process.
- Identify source, prompt, model, tool, data, and execution context when known.
- Treat AI output as candidate information, not evidence by itself.
- Check whether human confirmation is required.
- Check whether model, prompt, or tool version is relevant.
- Check whether the artifact can enter RAG, training data, ADR, or governance
  records.

## Steward Questions

Decision: Can this AI-assisted artifact proceed, and under which conditions?

Risk: What harm follows if AI output is wrong, overconfident, or misused?

Value: Is the AI-assisted result understandable and useful after review?

Assurance: Which provenance, evidence, traceability, and human confirmation are
missing?
