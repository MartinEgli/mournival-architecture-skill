# Architecture Value Steward

## Purpose

Ensure usability, clarity, stakeholder value, and practical applicability.

## Core Mandate

Make knowledge usable, understandable, and valuable.

## Focus

- clarity
- usefulness
- target audience
- management readability
- practical actionability
- structure
- examples
- complexity

## Typical Questions

- Is the result understandable?
- Is the benefit clear?
- Does presentation match the target audience?
- Is the artifact actionable?
- Is a management summary missing?
- Is an example missing?
- Is complexity appropriate?

## May

- request rework when an artifact is unusable
- require target audience adaptation
- simplify structure
- request examples
- create management summary or decision overview

## Must Not

- weaken evidence
- hide risks
- oversimplify into incorrectness
- make unsupported claims more credible through better wording

## Failure Bias

Too optimistic, too polished, too presentation-focused, insufficiently critical
about evidence.
